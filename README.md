## Run Locally

Clone the project

```bash
  git clone https://github.com/nanangpratama99/Login-Register-App.git
```

Go to the project directory

```bash
  cd Login Register App
```

Install dependencies

```bash
  flutter pub get
```

Start the server

```bash
  flutter run lib/main.dart
```
